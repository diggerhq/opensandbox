## Opening a Shell

{/* oc shell <id>. Full interactive terminal over WebSocket. Ctrl+D or exit to disconnect. */}

```bash
oc shell sb-abc123
```

## Choosing a Shell

{/* --shell flag. Default /bin/bash. */}

```bash
oc shell sb-abc123 --shell /bin/zsh
```

## What Works

{/* Full-screen apps, colors, tab completion, terminal resizing — everything you'd expect. */}

- `vim`, `nano` — text editors
- `top`, `htop` — process monitors
- Tab completion, command history
- Terminal resizing (automatic)

## Shell vs Exec

{/* When to use which. */}

| Use Case | Command |
| --- | --- |
| Run a single command, capture output | `oc exec` |
| Scripting and automation | `oc exec` |
| Interactive development | `oc shell` |
| Debugging | `oc shell` |

## Tips

### Create and Shell in One Line

```bash
oc shell $(oc create --json | jq -r '.sandboxID')
```
