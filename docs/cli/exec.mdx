## Capturing Output as JSON

{/* `--json` with `--wait` gives ProcessResult output for scripting. */}

```bash
RESULT=$(oc exec sb-abc123 --json --wait -- node -e 'console.log("hello")')
echo $RESULT | jq '.stdout'
```
