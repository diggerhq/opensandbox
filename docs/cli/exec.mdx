## Running a Command

{/* oc exec <id> -- <command>. How it works: command runs inside sandbox, stdout/stderr stream to terminal, CLI exits with same exit code. */}

```bash
oc exec sb-abc123 -- echo "Hello, World!"
```

## Working Directory & Environment

{/* --cwd and --env flags. When to use them. */}

```bash
oc exec sb-abc123 --cwd /app --env NODE_ENV=production -- npm run build
```

## Capturing Output as JSON

{/* --json flag gives ProcessResult object. Pipe to jq for scripting. */}

```bash
RESULT=$(oc exec sb-abc123 --json -- node -e 'console.log("hello")')
echo $RESULT | jq '.stdout'
```
