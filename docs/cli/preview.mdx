## Create a Preview URL

```bash
oc preview create sb-abc123 --port 3000
```

With custom domain:

```bash
oc preview create sb-abc123 --port 8080 --domain preview.example.com
```

## List Preview URLs

```bash
oc preview list sb-abc123
```

## Delete a Preview URL

```bash
oc preview delete sb-abc123 3000
```
