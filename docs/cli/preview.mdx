## Exposing a Port

{/* oc preview create. Public HTTPS URL that proxies to a sandbox port. */}

```bash
oc preview create sb-abc123 --port 3000
```

## Sharing a Dev Server

{/* Start a server inside sandbox, expose it, share the URL. */}

```bash
oc exec sb-abc123 -- bash -c 'cd /app && npm run dev &'
oc preview create sb-abc123 --port 3000

# Get the URL
oc preview list sb-abc123 --json | jq -r '.[0].hostname'
```
