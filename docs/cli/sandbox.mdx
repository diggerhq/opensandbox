## Creating a Sandbox

{/* oc create (shortcut for oc sandbox create). Template, resources, env vars. */}

```bash
oc create
oc create --timeout 600 --cpu 2 --memory 2048 --env NODE_ENV=production
```

## Listing & Inspecting

{/* oc ls (shortcut). Table output. oc sandbox get for details. */}

```bash
oc ls
oc sandbox get sb-abc123
```

## Hibernation & Wake

{/* Save state and stop paying. Resume exactly where you left off. */}

```bash
oc sandbox hibernate sb-abc123

# ... hours later ...
oc sandbox wake sb-abc123
oc shell sb-abc123  # picks up where you left off
```

## Common Patterns

### Create and Shell In

```bash
oc shell $(oc create --json | jq -r '.sandboxID')
```

### Filter Running Sandboxes

```bash
oc ls --json | jq '.[] | select(.status == "running") | .sandboxID'
```
