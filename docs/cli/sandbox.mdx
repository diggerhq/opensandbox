## Creating a Sandbox

{/* oc create (shortcut for oc sandbox create). Timeout, resources, env vars, metadata.

*/}

```bash

oc create --timeout 600 --cpu 2 --memory 2048 --env NODE_ENV=production
```

## Hibernation & Wake

{/* Save state and stop paying. Resume the same sandbox later; keep the wording practical rather than over-promising exact process continuity. */}

```bash
oc sandbox hibernate sb-abc123

# ... hours later ...
oc sandbox wake sb-abc123
oc shell sb-abc123  # picks up where you left off
```

## Common Patterns

### Create and Shell In

```bash
oc shell $(oc create --json | jq -r '.sandboxID')
```

### Filter Running Sandboxes

```bash
oc ls --json | jq '.[] | select(.status == "running") | .sandboxID'
```
