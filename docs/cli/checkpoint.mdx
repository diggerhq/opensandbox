## Create a Checkpoint

```bash
oc checkpoint create sb-abc123 --name my-checkpoint
```

## List Checkpoints

```bash
oc checkpoint list sb-abc123
```

## Restore

```bash
oc checkpoint restore sb-abc123 cp-7f3a1b2c
```

## Spawn (Fork)

```bash
oc checkpoint spawn cp-7f3a1b2c
```

## Delete

```bash
oc checkpoint delete sb-abc123 cp-7f3a1b2c
```
