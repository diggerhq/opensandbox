## Sandbox

### Class Methods

#### `await Sandbox.create(template?, timeout?, api_key?, api_url?, envs?, metadata?): Sandbox`

{/* No cpuCount/memoryMB — use HTTP API for custom resources */}

#### `await Sandbox.connect(sandbox_id, api_key?, api_url?): Sandbox`

#### `await Sandbox.create_from_checkpoint(checkpoint_id, ...): Sandbox`

#### `await Sandbox.create_checkpoint_patch(checkpoint_id, script, description?, ...): dict`

#### `await Sandbox.list_checkpoint_patches(checkpoint_id, ...): list[dict]`

#### `await Sandbox.delete_checkpoint_patch(checkpoint_id, patch_id, ...): None`

### Context Manager

```python
async with await Sandbox.create() as sandbox:
    # sandbox auto-kills on exit
    pass
```

{/* No resume param. No on_exit, on_scrollback_end. */}

### `await sandbox.agent.attach(session_id, on_event?, on_error?): AgentSession`

### `await sandbox.agent.list(): list[dict]`

### AgentSession

{/* session_id, collect_events(), wait(), send_prompt(), interrupt(), configure(), kill(), close() */}

#### `session.collect_events(): list[AgentEvent]`

{/* Python-unique: returns all events since last call */}

#### `await session.wait(): int`

{/* Python-unique: waits for session to complete, returns exit code */}

### AgentEvent

{/* Dataclass: type, data. Dict-like access: event['type'], event.get('content'). */}

{/* ProcessResult: exit_code, stdout, stderr */}
