## WebSocket Binary Protocol

{/* Exec and PTY sessions use binary WebSocket frames.
  Input: prefix 0x00 + stdin data.
  Output: 0x01 (stdout), 0x02 (stderr), 0x03 (exit code, 4-byte BE int32), 0x04 (scrollback end marker).
  On connect: server replays scrollback buffer, sends 0x04, then live-streams.
*/}
