## API Reference

<Note>
  Templates is a standalone class, not a property on Sandbox. TypeScript exports `Templates` (plural), Python exports `Template` (singular).
</Note>

### Build Template

{/* templates.build(name, dockerfile) / Template.build(name, dockerfile). ParamField. */}

### List Templates

{/* templates.list() / Template.list(). Returns TemplateInfo[]. */}

### Get Template

{/* templates.get(name) / Template.get(name). */}

### Delete Template

{/* templates.delete(name) / Template.delete(name). */}

### Using with Sandbox.create

{/* Sandbox.create({ template: 'my-template' }) */}
