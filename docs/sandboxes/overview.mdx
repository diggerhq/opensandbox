## Hibernation & Wake

{/* Like closing a laptop lid: sandbox identity stays the same and the platform attempts a fast snapshot-based resume.
  - sandbox.hibernate(), sandbox.wake() — TS SDK
  - Usually resumes quickly, no cost while hibernated
  - Auto-triggered on idle timeout
  - Difference from checkpoints: hibernation = transparent resume; checkpoints = named snapshots you fork from
  - Note: Python SDK doesn't have hibernate/wake. HTTP API: POST /api/sandboxes/:id/hibernate, wake. CLI: oc sandbox hibernate, wake.
*/}

<Note>
  `hibernate()` and `wake()` are available in the TypeScript SDK and CLI. Python SDK support is planned — use the HTTP API directly for now.
</Note>
