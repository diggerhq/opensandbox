---
title: 'Create Cancellations'
api: 'POST /api/capacity/cancellations'
---

<Info>
  **Coming soon.** Reserved capacity is not yet available. See the
  [Reserved Capacity overview](/reserved-capacity/overview).
</Info>

Cancel one or more capacity blocks by ID. Atomic all-or-nothing — if any
block in the request cannot be cancelled, none are cancelled. No
`skipIneligible` mode; retry with a refreshed ID list instead.

Send an `Idempotency-Key` header so retries return the original result.

<ParamField body="blockIds" type="string[]" required>
  Block UUIDs to cancel. Must belong to the authenticated org and be
  currently cancellable. Duplicates are de-duplicated server-side.
</ParamField>
