---
title: 'Cancel Block'
api: 'DELETE /api/capacity/blocks/{blockId}'
---

<Info>
  **Coming soon.** Reserved capacity is not yet available. See the
  [Reserved Capacity overview](/reserved-capacity/overview).
</Info>

Cancel one capacity block by ID. Equivalent to a one-block
`POST /api/capacity/cancellations` call.

<ParamField path="blockId" type="string" required>
  Block UUID. Must belong to the authenticated org and be currently cancellable.
</ParamField>

<ResponseExample>
```http 204
HTTP/1.1 204 No Content
```

```json 409 blocks_not_cancellable
{
  "error": "blocks_not_cancellable",
  "blocks": [
    { "blockId": "9f67b8f7-7b91-4d2d-b1cb-19d0d0a14562", "reason": "not_cancellable" }
  ]
}
```
</ResponseExample>
