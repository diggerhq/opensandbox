---
title: 'List Agents'
api: 'GET /v1/agents'
---

List all agents owned by the authenticated API key.

**CLI equivalent:** `oc agent list`

<ResponseExample>
```json 200
{
  "agents": [
    {
      "id": "my-hermes",
      "display_name": "my-hermes",
      "core": "hermes",
      "channels": ["telegram"],
      "packages": ["gbrain"],
      "secret_store": "agent:my-hermes",
      "config": null,
      "created_at": "2026-04-09T10:30:00Z",
      "updated_at": "2026-04-09T10:30:00Z"
    }
  ]
}
```
</ResponseExample>
