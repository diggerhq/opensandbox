---
title: 'Read File'
api: 'GET /api/sandboxes/{id}/files'
---

Returns file content as plain text (not JSON). Files are streamed in 256KB chunks. Maximum file size: **2 GB**.

<ParamField path="id" type="string" required>
  Sandbox ID
</ParamField>

<ParamField query="path" type="string" required>
  Absolute path to the file
</ParamField>

<ResponseExample>
```text 200
Hello, World!
```
</ResponseExample>
